# Classic JRPG Map References

This folder turns the supplied NES/SNES reference list into a curated Atlas Academy source catalog.

## Contents

- `catalog.json` contains machine-readable records for 24 high-yield maps spanning overworlds, towns, castles, dungeons, caves, and hybrid layouts.
- `catalog.md` is the human-readable learning guide.
- `../../academy/case-studies/comparative-jrpg-corpus-001.md` records the first direct visual study pass.
- `../../academy/knowledge/classic-jrpg-feel.md` converts the shared findings into a player-facing design doctrine for future Academy reviews.

## Use Rules

- Every entry is a `comparative_jrpg_reference` and a `candidate`, not an approved or gold-standard source.
- Use these references for comparative context, composition analysis, historical context, and pattern-extraction questions only.
- Do not infer RPG Maker implementation details from them.
- Do not reproduce layouts, tiles, sprites, names, or game content in production.
- Cite the direct source URL and the access date (`2026-07-12`) in any Academy observation.
- Images remain on their source sites. AtlasStudio governance requires separate licensing clearance before proprietary map images or tilesets may be copied into this repository.
