# Reference Materials

This directory contains read-only reference material used by AtlasStudio during planning, review, and implementation.

## Purpose

Reference materials help Atlas and implementation agents understand how external systems are structured without becoming part of the project's canon or implementation.

Examples include:

- RPG Maker MZ sample projects
- Engine documentation
- Tileset references
- Style guides
- Overworld reference images

## Rules

- These files are reference-only.
- They are not part of The Last Sword Protocol canon.
- They must not be modified unless intentionally updating the reference library.
- They must not be copied directly into production without an approved work order.

Atlas may analyze these resources to learn implementation patterns, quality standards, and engine conventions.
