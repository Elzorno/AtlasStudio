# RPG Maker MZ Sample Project Reference

## Purpose

This directory contains one or more RPG Maker MZ sample/demo projects used as implementation references.

The purpose is to help AtlasStudio and AI agents understand:

- high-quality map composition
- official map density and decoration
- event organization
- transfer placement
- tileset usage
- default passability behavior
- RPG Maker JSON structure

## This Is NOT Canon

Nothing in this directory is part of The Last Sword Protocol.

Names, locations, events, dialogue, and maps must never be treated as story canon.

## Allowed Uses

Agents may:

- inspect JSON files
- inspect map structure
- inspect Tilesets.json
- inspect MapInfos.json
- learn event organization
- learn layering techniques
- study map dimensions
- study environmental storytelling
- compare implementation quality against official examples

## Prohibited Uses

Agents must not:

- copy sample maps directly into The Last Sword Protocol
- copy dialogue or story
- overwrite production assets with sample assets
- modify the sample project
- treat sample content as gameplay requirements

## AtlasStudio Quality Goal

The visual quality of The Last Sword Protocol should be comparable to polished RPG Maker MZ sample maps while remaining faithful to the game's own story, regional overworld design, and JRPG Design Bible.

## Recommended Future Tooling

Future Atlas tools may automatically analyze these sample maps to establish quality baselines for:

- decoration density
- landmark usage
- event density
- passability validation
- building spacing
- map composition
- environmental storytelling

These analyses should inform implementation, not constrain creativity.
