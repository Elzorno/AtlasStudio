# Schema-First MZ Mapgen Prototype

Working JavaScript prototype for deterministic intent-to-topology-to-semantic-grid-to-MZ compilation. See `docs/pipeline_design.md` for assumptions and production gates.

```sh
npm test
npm run generate
npm run artwork
```

`npm run artwork` produces a disposable Ashford-styled rural-village review candidate using real Outside autotiles and approved TileAssemblies. It does not modify or promote a production Game map.
